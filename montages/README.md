# Montage audio files

Finished gathering montages live here as plain `.mp3` files, served as static
assets by Netlify. Naming convention: `YYYY-MM-DD-location-slug.mp3`, e.g.
`2026-08-25-atlanta.mp3`.

To add a new one: drop the file in this folder, then add a matching entry to
`../montages-data.js` (see the comment there for the shape). Commit and push
to deploy.
